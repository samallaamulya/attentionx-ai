# 🔥 AttentionX AI - Viral Intelligence Engine

## 🚀 Overview
AttentionX AI is an intelligent video processing system that converts long videos into short, high-engagement viral clips using AI-style analysis. It automatically detects potential viral moments, generates multiple clips, ranks them using a virality scoring engine, and suggests optimized captions for social media platforms like Instagram Reels, YouTube Shorts, and TikTok.

---

## 🎯 Problem Statement
Long videos have low engagement on short-form platforms. Manually finding the best moments for clipping is time-consuming and inefficient for content creators.

---

## 💡 Solution
AttentionX AI automates the entire process:

- Extracts multiple short clips from long videos  
- Assigns a virality score to each clip  
- Ranks clips based on engagement potential  
- Highlights the best-performing clip  
- Generates viral captions and hook suggestions  

---

## ⚙️ Features

### 🎬 Video Processing
- Upload videos in MP4, MOV, AVI formats  
- Automatic video loading and processing  

### 🧠 AI Virality Engine
- Generates multiple candidate clips  
- Simulated engagement prediction system  
- Ranks clips based on virality score  

### 🔥 Content Optimization
- Auto-generated viral captions  
- Hook suggestions for engagement  
- Best clip selection system  

### 📊 Analytics Dashboard
- Average virality score  
- Best clip highlight  
- AI-style insights report  

---

## 🛠️ Tech Stack
- Python  
- Streamlit (UI framework)  
- MoviePy (Video processing)  
- Random-based AI simulation model (virality scoring logic)
